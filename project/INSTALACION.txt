1) En archivo local/app/helpers.php configurar constantes
		WEB_URL a direccion web del servidor
		WEB_ROOT a la sub ruta a mostrar en la url web del sistema
		PROJECT_FOLDER a la ruta raiz del proyecto a partir del $_SERVER["DOCUMENT_ROOT"]

2) En archivo local/config/database.php, sub-campo 'mysql' del array a retornar, modificar atributos
	'host'		=> servidor de base de datos
	'database'	=> nombre de base de datos
	'username'	=> usuario de base de datos
	'password' 	=> contraseña de usuario anterior

	Para realizar conexion exitosa a base de datos

3) Cargar el script "admin_panel.sql" en la base de datos especificada anteriormente

4) Ahora el panel de administración debería correr sin problemas. Acceder con user:admin y password:123456

5) Si surgiese algún posible error, revisar el fichero local/storage/logs/laravel.log



RECORDAR:
	la ruta de los iconos (el de la parte superior izquierda y el del tab) en el archivo admin-panel-settings.php
